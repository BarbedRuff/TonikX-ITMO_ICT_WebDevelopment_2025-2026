# Лабораторная работа №1 - Сетевое программирование

## Описание

Данная лабораторная работа посвящена изучению основ сетевого программирования с использованием сокетов в Python. Работа включает в себя реализацию различных типов клиент-серверных приложений.

## Структура проекта

### Task 1 - UDP клиент-сервер
- **Папка**: `task1/`
- **Описание**: Простая реализация UDP сервера и клиента
- **Файлы**:
  - `server/server.py` - UDP сервер, принимающий сообщения от клиентов
  - `client/client.py` - UDP клиент для отправки сообщений серверу
  - `server/cfg.py`, `client/cfg.py` - конфигурационные файлы

### Task 2 - TCP клиент-сервер
- **Папка**: `task2/`
- **Описание**: Реализация TCP сервера и клиента
- **Файлы**:
  - `server/server.py` - TCP сервер
  - `client/client.py` - TCP клиент
  - `server/cfg.py`, `client/cfg.py` - конфигурационные файлы

### Task 3 - HTTP сервер
- **Папка**: `task3/`
- **Описание**: Простой HTTP сервер, отдающий HTML страницы
- **Файлы**:
  - `server/server.py` - HTTP сервер
  - `server/index.html` - HTML страница для отображения
  - `server/cfg.py` - конфигурационный файл

### Task 4 - Многопоточный TCP сервер
- **Папка**: `task4/`
- **Описание**: TCP сервер с поддержкой множественных клиентов
- **Файлы**:
  - `server/server.py` - многопоточный TCP сервер
  - `client/client.py` - TCP клиент
  - `server/cfg.py`, `client/cfg.py` - конфигурационные файлы

### Task 5 - Расширенный HTTP сервер
- **Папка**: `task5/`
- **Описание**: HTTP сервер с дополнительным функционалом
- **Файлы**:
  - `server/server.py` - расширенный HTTP сервер
  - `server/index.html` - HTML страница
  - `server/cfg.py` - конфигурационный файл

## Запуск приложений

### Для UDP/TCP серверов (Task 1, 2, 4):
1. Запустите сервер:
   ```bash
   cd taskX/server/
   python server.py
   ```
2. В отдельном терминале запустите клиент:
   ```bash
   cd taskX/client/
   python client.py
   ```

### Для HTTP серверов (Task 3, 5):
1. Запустите сервер:
   ```bash
   cd taskX/server/
   python server.py
   ```
2. Откройте браузер и перейдите по адресу, указанному в консоли

## Технологии

- **Python 3** - основной язык программирования
- **Socket** - библиотека для работы с сокетами
- **Threading** - для многопоточности (Task 4)
- **HTTP** - протокол для веб-серверов (Task 3, 5)
