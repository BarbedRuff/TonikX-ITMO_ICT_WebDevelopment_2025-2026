# Restaurant Management Frontend

Vue.js фронтенд для системы управления рестораном с Vuetify UI.

## Технологии

- **Vue 3** - основной фреймворк
- **Vuetify** - Material Design компоненты
- **Vue Router** - маршрутизация
- **Axios** - HTTP клиент
- **Vite** - сборщик

## Структура проекта

```
src/
├── views/           # Страницы приложения
│   ├── Login.vue    # Авторизация
│   ├── Register.vue # Регистрация
│   ├── Dashboard.vue # Главная панель
│   ├── Employees.vue # Управление сотрудниками
│   ├── Ingredients.vue # Управление ингредиентами
│   ├── Dishes.vue   # Управление блюдами
│   ├── Tables.vue   # Управление столами
│   ├── Orders.vue   # Управление заказами
│   └── Positions.vue # Управление должностями
├── services/
│   └── api.js       # API клиент
├── stores/
│   └── auth.js      # Управление авторизацией
├── router/
│   └── index.js     # Конфигурация роутов
├── App.vue          # Корневой компонент
└── main.js          # Точка входа
```

## Основные компоненты

### Dashboard.vue
Главная панель с навигационными карточками для доступа к разделам системы.

### Login.vue / Register.vue
Формы авторизации и регистрации с валидацией полей.

### Employees.vue
- Таблица сотрудников с пагинацией
- Модальное окно создания/редактирования
- Фильтрация по категориям (повар, официант)
- Связь с должностями

### Ingredients.vue
- Список ингредиентов с индикаторами запасов
- Кнопка "Низкий запас" для фильтрации
- Управление поставщиками и типами

### Dishes.vue
- Каталог блюд с автоматическим расчетом цен
- Группировка по типам блюд
- Информация об ингредиентах

### Tables.vue
- Схема столов с статусами
- Быстрое изменение статуса (свободен/занят)
- Назначение официантов

### Orders.vue
- Список заказов с фильтрацией по статусам
- Создание сложных заказов с несколькими блюдами
- Управление жизненным циклом заказа
- Расчет общей стоимости

### Positions.vue
- Справочник должностей
- Настройка минимальных зарплат

## API интеграция

### services/api.js
Централизованный HTTP клиент с:
- Автоматическим добавлением токена авторизации
- Базовым URL для API
- Методами для всех сущностей системы

### stores/auth.js
Композабл для управления состоянием авторизации:
- Хранение токена в localStorage
- Автоматическая проверка авторизации
- Методы login/logout/register

## Роутинг

Защищенные маршруты требуют авторизации:
- `/` - Dashboard (защищен)
- `/login` - Авторизация
- `/register` - Регистрация
- `/employees` - Сотрудники (защищен)
- `/ingredients` - Ингредиенты (защищен)
- `/dishes` - Блюда (защищен)
- `/tables` - Столы (защищен)
- `/orders` - Заказы (защищен)
- `/positions` - Должности (защищен)

## UI/UX особенности

### Vuetify компоненты
- `v-data-table` для списков с сортировкой и пагинацией
- `v-dialog` для модальных окон
- `v-form` с валидацией полей
- `v-chip` для статусов и индикаторов
- `v-card` для группировки контента

### Цветовая схема
- **Синий** - сотрудники
- **Зеленый** - ингредиенты
- **Оранжевый** - блюда
- **Голубой** - столы
- **Красный** - заказы
- **Фиолетовый** - должности

### Адаптивность
- Responsive дизайн для мобильных устройств
- Гибкая сетка с `v-row`/`v-col`
- Адаптивные таблицы

## Функциональность

### Авторизация
- JWT токены с автоматическим обновлением
- Защита роутов от неавторизованного доступа
- Автоматический редирект после логина

### CRUD операции
- Создание, чтение, обновление, удаление для всех сущностей
- Валидация форм на клиенте
- Обработка ошибок API

### Фильтрация и поиск
- Фильтр ингредиентов по низким запасам
- Фильтр заказов по статусам
- Поиск в таблицах

### Интерактивность
- Быстрое изменение статусов столов и заказов
- Drag & drop для некоторых операций
- Подтверждение удаления

## Запуск

```bash
npm install
npm run dev
```

Приложение будет доступно на `http://localhost:5173`
