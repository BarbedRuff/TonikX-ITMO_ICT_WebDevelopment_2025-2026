# API Эндпоинты Warriors Project

## Базовые эндпоинты

### 1. Просмотр всех воинов
- **URL**: `GET /war/warriors/`
- **Описание**: Возвращает список всех воинов
- **Ответ**: JSON с массивом воинов

### 2. Создание профессии
- **URL**: `POST /war/profession/create/`
- **Описание**: Создает новую профессию
- **Тело запроса**: 
```json
{
  "profession": {
    "title": "Название профессии",
    "description": "Описание профессии"
  }
}
```

### 3. Просмотр всех скилов
- **URL**: `GET /war/skills/`
- **Описание**: Возвращает список всех скилов

### 4. Создание скила
- **URL**: `POST /war/skills/create/`
- **Описание**: Создает новый скил
- **Тело запроса**:
```json
{
  "title": "Название скила"
}
```

## Практическое задание - новые эндпоинты

### 1. Вывод полной информации о всех войнах и их профессиях
- **URL**: `GET /war/warriors/professions/`
- **Описание**: Возвращает всех воинов с подробной информацией о их профессиях
- **Ответ**: JSON с воинами и их профессиями

### 2. Вывод полной информации о всех войнах и их скилах
- **URL**: `GET /war/warriors/skills/`
- **Описание**: Возвращает всех воинов с подробной информацией о их скилах и уровнях владения
- **Ответ**: JSON с воинами и их скилами

### 3. Вывод полной информации о войне (по id), его профессиях и скилах
- **URL**: `GET /war/warriors/{id}/`
- **Описание**: Возвращает подробную информацию о конкретном воине, включая профессию и скилы
- **Параметры**: `id` - ID воина
- **Ответ**: JSON с полной информацией о воине

### 4. Удаление война по id
- **URL**: `DELETE /war/warriors/{id}/delete/`
- **Описание**: Удаляет воина по указанному ID
- **Параметры**: `id` - ID воина
- **Ответ**: Статус 204 No Content при успешном удалении

### 5. Редактирование информации о войне
- **URL**: `PUT/PATCH /war/warriors/{id}/update/`
- **Описание**: Обновляет информацию о воине
- **Параметры**: `id` - ID воина
- **Тело запроса**:
```json
{
  "name": "Новое имя",
  "race": "d",
  "level": 10,
  "profession": 1
}
```

### 6. Комбинированный эндпоинт для управления воином
- **URL**: 
  - `GET /war/warriors/{id}/manage/` - получить информацию
  - `PUT/PATCH /war/warriors/{id}/manage/` - обновить информацию
  - `DELETE /war/warriors/{id}/manage/` - удалить воина
- **Описание**: Универсальный эндпоинт для работы с конкретным воином

## Примеры использования

### Создание воина
```bash
curl -X POST http://localhost:8000/war/warriors/ \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Иван Воин",
    "race": "d",
    "level": 5,
    "profession": 1
  }'
```

### Получение воина со всей информацией
```bash
curl http://localhost:8000/war/warriors/1/
```

### Обновление воина
```bash
curl -X PATCH http://localhost:8000/war/warriors/1/update/ \
  -H "Content-Type: application/json" \
  -d '{
    "level": 10
  }'
```

### Удаление воина
```bash
curl -X DELETE http://localhost:8000/war/warriors/1/delete/
```

## Типы рас (race)
- `s` - student
- `d` - developer  
- `t` - teamlead
