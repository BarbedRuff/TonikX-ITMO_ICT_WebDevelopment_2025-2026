# Лабораторная работа №2 - Веб-приложение "Турагентство"

## Описание

Данная лабораторная работа представляет собой веб-приложение для управления турагентством, разработанное с использованием Django. Приложение позволяет пользователям просматривать туры, делать резервирования и оставлять отзывы.

## Функциональность

### Основные возможности:
- **Просмотр туров** - каталог доступных туров с подробной информацией
- **Регистрация и авторизация** - система пользователей
- **Резервирование туров** - возможность забронировать тур
- **Управление резервированиями** - просмотр, редактирование и отмена бронирований
- **Система отзывов** - возможность оставлять отзывы о турах
- **Аналитика** - статистика продаж по странам

### Модели данных:
- **Country** - страны назначения
- **TourAgency** - турагентства
- **Tour** - туры с описанием, датами и ценами
- **Reservation** - резервирования пользователей
- **Review** - отзывы пользователей о турах

## Установка и запуск

### Локальный запуск:

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Выполните миграции:
   ```bash
   python manage.py migrate
   ```

3. Загрузите тестовые данные:
   ```bash
   python manage.py loaddata tours/fixtures/test_data.json
   ```

4. Создайте суперпользователя:
   ```bash
   python manage.py createsuperuser
   ```

5. Запустите сервер:
   ```bash
   python manage.py runserver
   ```

### Запуск через Docker:

1. Запустите контейнеры:
   ```bash
   docker-compose up --build
   ```

2. Приложение будет доступно по адресу: http://localhost:8000

## Основные URL маршруты

- `/` - главная страница со списком туров
- `/tour/<id>/` - детальная информация о туре
- `/reserve/<id>/` - резервирование тура
- `/my-reservations/` - мои резервирования
- `/login/` - авторизация
- `/register/` - регистрация
- `/sales-by-country/` - статистика продаж по странам
- `/admin/` - административная панель
