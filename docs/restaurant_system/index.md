# Ресторанная система

Полнофункциональное приложение для управления рестораном, состоящее из фронтенда на Vue.js и бэкенда на Django REST API.

## Обзор системы

Ресторанная система представляет собой комплексное решение для автоматизации работы ресторана, включающее:

- Управление сотрудниками и должностями
- Учет ингредиентов и блюд
- Управление столами и заказами
- Систему авторизации и разграничения прав доступа

## Архитектура

Система построена на основе современной архитектуры с разделением на фронтенд и бэкенд:

- **Фронтенд**: Vue.js приложение с компонентным подходом и Vuetify UI
- **Бэкенд**: Django REST API с использованием Django REST Framework

## Компоненты системы

### Фронтенд

Интерфейс пользователя разработан с использованием Vue.js и включает следующие основные разделы:

- Панель управления (Dashboard)
- Управление сотрудниками
- Управление ингредиентами
- Управление блюдами
- Управление столами
- Управление заказами
- Управление должностями

### Бэкенд

API сервер на Django REST Framework предоставляет следующие основные эндпоинты:

- Авторизация и регистрация
- CRUD операции для всех сущностей системы
- Специализированные эндпоинты для бизнес-логики

## Технологии

### Фронтенд
- **Vue 3** - основной фреймворк
- **Vuetify** - Material Design компоненты
- **Vue Router** - маршрутизация
- **Axios** - HTTP клиент
- **Vite** - сборщик

### Бэкенд
- **Django** - веб-фреймворк
- **Django REST Framework** - API фреймворк
- **Djoser** - авторизация
- **PostgreSQL** - база данных
