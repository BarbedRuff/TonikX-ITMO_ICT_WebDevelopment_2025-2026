# ITMO ICT Веб-разработка 2025-2026

Добро пожаловать в документацию курса по веб-разработке в Университете ИТМО.

## Лабораторные работы

Эта документация охватывает следующие лабораторные работы:

1. [Лабораторная работа 1](lab1/index.md) - Сетевое программирование с использованием Python sockets
2. [Лабораторная работа 2](lab2/index.md) - Веб-приложение на Django "Туристическое агентство"
3. [Ресторанная система](restaurant_system/index.md) - Полнофункциональное приложение с фронтендом на Vue.js и бэкендом на Django REST API

## Структура репозитория

Репозиторий организован следующим образом:

- `lab1/` - Содержит код для Лабораторной работы 1 (Сетевое программирование)
- `lab2/` - Содержит код для Лабораторной работы 2 (Туристическое агентство)
- `restaurant_system/` - Содержит код для проекта Ресторанной системы

## Начало работы

Для изучения лабораторных работ используйте меню навигации слева для просмотра различных разделов.

Каждая лабораторная работа имеет собственную документацию с подробными объяснениями задач, деталями реализации и инструкциями по запуску кода.
